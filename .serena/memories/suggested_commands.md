# Suggested Commands

## Installation
- `npm install`: Install project dependencies.

## Development
- `npm run dev`: Start the development server (Vite, auto-opens browser to `http://localhost:5173`).

## Building
- `npm run build`: Build the project for production (`dist/` directory).
- `npm run preview`: Preview the production build locally.

## Code Quality
- `npm run lint`: Run ESLint to check for code quality and style issues.
- `npm run type-check`: Run TypeScript compiler to check for type errors.

## Testing
- `npm run test`: Run tests (uses Jest and React Testing Library).
- `npm run test:watch`: Run tests in watch mode.
