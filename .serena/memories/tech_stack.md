# Tech Stack

## Core Dependencies

- **React 18.2.0**: JavaScript library for building user interfaces.
- **React DOM 18.2.0**: React rendering for web browsers.
- **React Router DOM 6.8.0**: Declarative routing for React applications.
- **TypeScript 5.0.2**: Static type checking for JavaScript.
- **Vite 4.4.5**: Next-generation frontend build tool.
- **Tailwind CSS 3.3.5**: Utility-first CSS framework.
- **Shadcn/ui**: For reusable UI components (e.g., Button).

## Other Dependencies & Tools

- **PostCSS 8.4.31** with **Autoprefixer 10.4.16**: CSS transformation and vendor prefix automation.
- **ESLint 8.45.0** (with `@typescript-eslint/eslint-plugin`, `@typescript-eslint/parser`, `eslint-plugin-react-hooks`, `eslint-plugin-react-refresh`): JavaScript/TypeScript linting tool.
- **class-variance-authority**, **clsx**, **lucide-react**, **tailwind-merge**, **tailwindcss-animate**: Utilities for styling and UI components (likely related to Shadcn/ui).

## Development Environment

- **Node.js** (types via `@types/node`)
- **Vite** (plugin via `@vitejs/plugin-react`)
- **VS Code** (implied by typical TypeScript/React setup)
